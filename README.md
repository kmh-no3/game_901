# GameTemplate

シンプルなHTML5キャンバスゲームのテンプレートです。

## 🎮 プレイ

### 🌐 オンラインでプレイ

**[▶️ ゲームをプレイする](https://kmh-no3.github.io/game_901/)**

> **📝 注意**: GitHub Pagesが有効になっている場合、上記のURLでゲームにアクセスできます。
> 
> GitHub Pagesを有効にするには、リポジトリの Settings → Pages で「/ (root)」を選択してください。

## 📁 ファイル構成

- `index.html` - メインのHTMLファイル
- `game.js` - ゲームロジック
- `reimu.png` - プレイヤーキャラクター画像
- `marisa.png` - 敵キャラクター画像
- `moon.png` - 月の背景画像
- `castle.png` - 城の背景画像

## 🎯 遊び方

1. キーボードのキーを押すとキャラクターがジャンプします
2. 敵を避けてスコアを稼ぎましょう
3. 敵に当たるとゲームオーバーです
