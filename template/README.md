# GameTemplate

シンプルなHTML5キャンバスゲームのテンプレートです。

## ファイル構成

- `template.html` - メインのHTMLファイル
- `template.js` - ゲームロジック
- `reimu.png` - プレイヤーキャラクター画像
- `marisa.png` - 敵キャラクター画像

## 遊び方

1. `template.html` をブラウザで開く
2. キーボードのキーを押すとキャラクターがジャンプします
3. 敵を避けてスコアを稼ぎましょう

## 技術スタック

- HTML5 Canvas
- JavaScript (Vanilla)
